# Frontend

Vue 3 + Vite + TypeScript UI for StoerGeler (Naive UI + FullCalendar).

## Local dev

```bash
npm install
npm run dev
```

Open `http://localhost:5173`. The dev server proxies `/api` to `http://localhost:8001` (see `vite.config.ts`).

## Build

```bash
npm run build
```

Build output: `dist/`
